# 1121538_徐霈綺_房貸計算器

這是一個基於 C# Windows Forms 開發的房貸試算工具。除了具備精確的「本息平均攤還」計算功能外，更針對使用者體驗設計了**響應式視窗縮放**、**自定義圓角 UI** 以及**全鍵盤操作支援**，讓房貸試算變得直觀且現代化。

## 🌟 核心特色

* **彈性輸入**：支援自備款以「百分比 (%)」或「固定金額」輸入，適應不同購屋場景。
* **寬限期設定**：完整模擬寬限期內「只付利息」與寬限期後「本息攤還」的金額落差。
* **響應式介面 (Responsive UI)**：自定義縮放邏輯，視窗拉伸時控制項位置與字體大小會自動等比例調整。
* **全鍵盤操作**：支援 `Tab` 切換與 `Alt + 快速鍵` (如 Alt+C 進行計算)，大幅提升操作效率。
* **歷史紀錄系統**：自動保存計算紀錄，點擊即可快速還原之前的參數設定，方便比對不同方案。
* **多幣值支援**：內建常用幣值切換，亦可自定義輸入。

### 操作步驟
1.  **輸入房屋總價**：以「萬元」為單位輸入。
2.  **設定自備款與幣值**：選擇輸入方式並指定貨幣。
3.  **填寫貸款參數**：輸入年利率、貸款年限及寬限期（選填）。
4.  **執行計算**：點擊 **計算(C)** 或使用快捷鍵 `Alt + C`。
5.  **結果判讀**：
    * **貸款總金額**：房屋總價扣除自備款後的餘額。
    * **每月應繳金額**：區分寬限期前與寬限期後的支出。
    * **首期明細**：即時掌握第一個月的本利佔比。

## 🛠 技術細節

### 響應式布局解決方案
為解決 WinForms 視窗縮放時控制項跑位的問題，本專案在 `Form_Load` 時利用 `Dictionary` 結構紀錄所有控制項的原始位置 (`Rectangle`) 與字體大小。在 `Resize` 事件觸發時，透過動態計算比例因子 (Ratio) 即時重繪所有組件，確保 UI 始終保持比例正確。

### 計算公式：本息平均攤還法
本程式計算邏輯參考[內政部不動產資訊平台](https://pip.moi.gov.tw/)，核心公式如下：

* **月利率 ($r$)** = 年利率 / 12 / 100
* **每月應繳金額 ($M$)**：
    $$M = P \times \frac{r(1+r)^n}{(1+r)^n - 1}$$
    *(其中 P 為貸款總本金，n 為總期數)*
